## О проекте
Этот репозиторий содержит **экзаменационное задание** по дисциплине **«Основы веб-технологий»**.  
Задача — разработать **сайт языковой школы**, где пользователь может:
- просматривать **курсы** и **репетиторов**,
- искать и фильтровать данные,
- **оформлять заявку** через модальное окно,
- управлять своими заявками в **личном кабинете** (просмотр / редактирование / удаление) через API. 

## Кратко: что было сделано
### Главная страница
- Шапка с навигацией
- Область уведомлений
- Блок “О школе”
- Блок преимуществ (с использованием Bootstrap Grid)
- Таблица курсов + поиск + **пагинация (по 5 элементов на странице)**
- Таблица репетиторов + фильтры (язык / уровень / опыт)
- Модальное окно “Оформление заявки” с расчётом стоимости
- Футер с контактами и соцсетями

### Личный кабинет
- Таблица заявок пользователя
- Действия: **Подробнее / Изменить / Удалить**
- Пагинация заявок (по 5 на страницу)
- Уведомления об успешных/ошибочных операциях

### Работа с API
- Используются AJAX-запросы (например, `fetch`) к CRUD API:
  - загрузка курсов/репетиторов/заявок,
  - создание заявки (POST),
  - редактирование (PUT),
  - удаление (DELETE).
- Авторизация выполняется через параметр `api_key` в строке запроса.
